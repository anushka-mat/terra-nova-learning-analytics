# Terra Nova Learning Analytics Dashboard

## Project Overview
This project analyzes employee engagement with Terra Nova Learning Academy courses across job roles, learning formats, cities, and course types. The goal was to identify participation gaps, understand which learning formats performed best, and recommend actions to improve course completion and employee learning outcomes.

## Business Problem
Terra Nova needed better visibility into how employees were using learning programs across different roles, locations, and course formats. The analysis focused on answering four key business questions:

- Which employee groups are engaging the most?
- Which courses are most popular?
- Which learning formats drive stronger completion?
- Where should Terra Nova improve access, participation, and learning outcomes?

## Dashboard Highlights
- In-Person learning had the strongest completion rate at approximately 93%.
- Online Live also performed well, with a completion rate around 82%.
- Async learning had the lowest completion rate at approximately 72%.
- Logistics Analysts and Warehouse Supervisors showed the highest participation.
- Baristas had the lowest participation and the lowest completion rate.
- Data Analytics for Operations, Leadership Foundations, and Transportation Management Systems were among the most enrolled courses.

## Key Insights
- Employees were more likely to complete training when learning was structured and synchronous.
- In-Person and Online Live formats were stronger than Async learning.
- Baristas appeared underserved by the current learning structure, likely due to scheduling, access, or course relevance issues.
- Learning preferences varied by location, suggesting that city-level learning strategies should be customized.
- PCA and clustering helped segment learners into profiles and identify groups with higher risk of lower completion.

## Business Recommendations
- Continue investing in In-Person and Online Live learning formats.
- Improve Async learning through reminders, shorter modules, progress tracking, and stronger manager support.
- Create shorter, mobile-friendly, role-specific modules for Baristas and other lower-participation groups.
- Localize learning strategies by city based on format preferences and participation patterns.
- Expand high-demand courses such as Data Analytics for Operations and Leadership Foundations.
- Use learner segmentation to target support toward at-risk groups and improve completion rates.

## Business Impact
This project translated employee learning data into actionable insights for training strategy, participation improvement, and workforce development. The dashboard and analysis helped identify where Terra Nova should invest, which employee groups need support, and how learning formats can be optimized to improve completion outcomes.
